# X-O (Tic Tac) game over TCP & UDP
Java application to play classic tic tac (X-O) game over tcp or udp connection.
Both players choose mode (tcp or udp) and port for the server (the payer who connects is the server).
Both players must choose the same port and mode.
Then one player enters the shown ip address of the other player in the same network.
